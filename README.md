# Fitness & Workout Tracker

A mobile application built with Flutter to help users discover and track various workout categories and exercises.

## Features

- **Workout Categories**: Browse through 9 different workout categories including:
  - Aerobic Exercises
  - Balance Exercises
  - Cardio Exercises
  - Flexibility Exercises
  - Mobility Training
  - Strength Workout
  - Stretching Exercises
  - Warm-Up Exercises
  - Yoga Exercises

- **Featured Workout**: Daily featured workout section with eye-catching design
- **Responsive Grid Layout**: Automatically adjusts the number of columns based on screen size
- **Interactive UI**: Add new workout categories with the floating action button
- **Search & Profile**: Placeholder icons for future search and user profile features

## Getting Started

### Prerequisites

- Flutter SDK (version ^3.11.1 or higher)
- Dart SDK (version ^3.11.1 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/Fitness_And_Workout_APP.git
```

2. Navigate to the project directory:
```bash
cd Fitness_And_Workout_APP
```

3. Install dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
